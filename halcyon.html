Eleven Lenses

Web Design

I was tasked with designing a record for the web, something showing the passing of time. So I decided to create a website for a literal record, an album, Deerhunter's Halcyon Digest. The site, “Eleven Lenses” is a listening platform for Halcyon Digest. It engages with visitors auditorily and visually. Play with it here on Github, but note there is no audio for copyright reasons.

On the site's homepage are eleven symbols, each drawn for a specific song from the album. Each symbol links to a different song page:

I found that while listening to the album, I started associating specific places of memory with songs. These places of memory are integral to the experience of the site.

On a song page, the song’s symbol serves as a lens through which a place from my memory can be explored:
